# HTML(5) Cheat Sheet

## Prelude

~~~~~~~ {#prelude .html .numberLines}
<!DOCTYPE html>
<html>
	<head>		
		<meta charset="UTF-8">
		<title>Title of the document</title>
	</head>
	<body>
		Content of the document......
	</body>
</html>
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

## CSS Link

~~~~~~~ {#css-link .html .numberLines}
<head>
	<link rel="stylesheet" type="text/css" href="theme.css">
</head> 
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
